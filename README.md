# cms-nano
Visualization of nanoAod

## To run:
  * Setup ROOTSYS environment from root master branch:
  
  `build/bin/thisroot.sh`

  * clone repository:
  
  `git clone https://github.com/alja/cms-nano.git`

  * start event display with make

`make`
